## 🔧 **Customization Guide**

### **Change Animation Speed**
In `assets/css/enhancements.css`:
```css
.scroll-reveal {
  animation: slideInUp 0.6s ease-out forwards;
  /* Change 0.6s to 0.3s, 1s, etc. */
}
```

### **Adjust Form Validation**
In `assets/js/enhancements.js`, modify the `validateField()` method:
```javascript
// Change minimum character requirements
isValid = value.length >= 2; // Name
isValid = value.length >= 10; // Message
```

### **Update EmailJS Service**
Replace the init key with your EmailJS credentials:
```javascript
emailjs.init("your_emailjs_user_id");
```
